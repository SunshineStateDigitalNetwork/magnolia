magnolia API
============

**magnolia** is a library in addition to a CLI tool. Classes may be imported into other programs and used separately
from the CLI.

+--------------------------+----------------------------------+
| magnolia.scenarios       | Parser and record classes        |
+--------------------------+----------------------------------+
| magnolia.source_resource | Result data class                |
+--------------------------+----------------------------------+
| magnolia.maps            | Default transformation functions |
+--------------------------+----------------------------------+
| magnolia.organizations   | Data provider abstraction class  |
+--------------------------+----------------------------------+
| magnolia.exceptions      | Magnolia specific exceptions     |
+--------------------------+----------------------------------+
